    Wat zijn gebruikersscripts?

    Gebruikersscripts stellen je in staat je browser naar je eigen wensen aan te passen. Wanneer je ze hebt geïnstalleerd, maken ze sites die je bezoekt automatisch beter, door aanvullende mogelijkheden, makkelijker gebruik of vervelende eigenschappen kunnen worden verwijderd. De gebruikersscripts op Greasy Fork zijn geschreven door andere gebruikers, en geplaatst om te delen met de rest van de wereld. Ze zijn gratis om te installeren, en makkelijk in gebruik.

    Stap 1: Installeer een gebruikersscriptmanager

    To use user scripts you need to first install a user script manager. Which user script manager you can use depends on which browser you use.

    • Chrome: Tampermonkey or Violentmonkey
    • Firefox: Greasemonkey, Tampermonkey, or Violentmonkey
    • Safari: Tampermonkey or Userscripts
    • Microsoft Edge: Tampermonkey
    • Opera: Tampermonkey or Violentmonkey
    • Maxthon: Violentmonkey
    • Dolphin: Tampermonkey
    • UC: Tampermonkey
    • AdGuard: (no additional software required)

    Stap 2: Installeer een gebruikersscript

    Bezoek deze site om een script te vinden die je wilt proberen. Hier is een voorbeeld van de meest populaire scripts:

    • 网课题搜答案 - 智慧树、超星、学习通、知到、慕课、上学吧答案大全
    • 智慧树网课助手(改) - 自动挂机看知到MOOC，支持屏蔽弹窗题目、自动切换下一节，章测试和考试支持自动答题，视频自动倍速播放、线路选择、默认静音等，解除各类功能限制，开放自定义参数

    Wanneer je een script hebt gevonden, klik je op de groene 'Installeer'-knop op de gebruikersscript pagina en de scriptmanager zal je vragen om de installatie te bevestigen.

    Stap 3: gebruik het script

    Ga naar de site die het script beïnvloed. Het script zou vanzelf z'n ding moeten doen. Na het uitproberen van het script ga je terug naar waar je het script hebt geïnstalleerd en kun je wat feedback achterlaten voor de auteur van het script.
